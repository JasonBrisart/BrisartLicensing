# Brisart Ecosystem Licensing

This file provides general licensing direction for Brisart ecosystem projects.

The Brisart ecosystem includes independently maintained software projects developed by Jason Brisart for local-first, auditable, maintainable, and long-term research environments.

Covered projects may include, but are not limited to:

- BrisartDevTools
- BrisartResearchArchive
- BrisartPreservationTools
- BrisartIdentityTools

and future Brisart software projects unless explicitly stated otherwise.

---

## Official Releases

Official Brisart releases are published through official Brisart-controlled repositories and distribution channels.

Users should refer to the official project repository and this BrisartLicensing repository for current licensing and ecosystem policy information.

---

## Current Licensing Reference

Each Brisart project may include a short licensing notice pointing to this repository.

This repository is intended to act as the central reference point for current licensing terms, ecosystem policies, participation information, and related documentation.

---

## Ecosystem Policy

Brisart software is developed first for the operational needs of the Brisart ecosystem and associated research environments.

The ecosystem prioritizes:

- Local-first operation
- Source availability
- Auditability
- Long-term maintainability
- Dependency-conscious development
- Research-oriented workflows
- Archival and preservation use cases
